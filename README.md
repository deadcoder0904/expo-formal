# expo-formal

> Using Expo with Formal to make the world's simplest forms in React Native

![App](./assets/app.png)

# Clone the repo

```bash
$ git clone git@github.com:deadcoder0904/expo-formal.git
```

# Install the dependencies

```bash
$ yarn
```

# Run the app

```bash
$ yarn start
```
